# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [0.1.0] - 2026-05-01

### Added
- Python runtime (`checkmate/`) implementing the chess-piece agent planning model.
- npm CLI (`bin/checkmate.js`) exposing `plan`, `doctor`, and skill installation commands via `npx`.
- Skill bundle (`skills/checkmate-agent-system/`) packaged as `dist/checkmate-agent-system-skill.zip` for installation into Claude and Codex skill roots.
